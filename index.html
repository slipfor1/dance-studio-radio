Dance Studio Radio

A music + practice tool for Mountain Home Ballroom dance studio.

About

Dance Studio Radio is an in-house web app for ballroom dance instruction and practice. It consolidates tempo-locked practice tracks, AI-generated music for every ballroom style, schedule-based class sessions, and Spotify playlist playback into a single tool used at the studio.

Features

  • Pitch-preserving tempo shift so any track can be slowed or sped up while practicing
  • BPM-locked metronome for every ballroom dance category
  • Session scheduling for competitive rounds and social practice mixes
  • Spotify integration for streaming the studio's curated playlists
  • Per-style playlist mapping with smart auto-detection

Powered by

  • Music streaming and playlist data: Spotify
  • Song tempo (BPM) data: GetSongBPM
